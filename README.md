to reproduce project, create a virtual environment using these steps

1 - Open the command palette (Ctrl+Shift+P) and select 'Python: Create Environment'

2 - Select 'Venv' environment type:

3 - select the latest version of the Python interpreter

4 - enter a name for the virtual environment (just use .venv, the default)

5 - select 'install project dependancies'

6 - select the 'requirements.txt' file

7 - done, now just wait for it to do its thing :D

 
8 - now 'run and debug' 

9 - select 'python debugger'

10 - select 'fask API'

done, now you can test the project yourself by visiting your local host (http://127.0.0.1:8000)
